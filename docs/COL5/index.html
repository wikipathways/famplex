COL5

A fibrillar collagen found widely distributed as a minor component in tissues that contain COLLAGEN TYPE I and COLLAGEN TYPE III. It is a heterotrimeric molecule composed of alpha1(V), alpha2(V) and alpha3(V) subunits. Several forms of collagen type V exist depending upon the composition of the subunits that form the trimer.
